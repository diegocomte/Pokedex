## Descrição do Projeto
A Pokédex emula o famoso jogo no qual se pode verificar todos os Pokémons e 'capturá-los'.

## Instalação
```
$ git clone https://github.com/diegocomte/Pokedex.git
$ npm install
$ npm start
```
